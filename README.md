Ecosystem Simulation

This project simulates a 2D ecosystem using JavaScript and HTML5 Canvas.
The world is made from hexagonal tiles with different biomes.
Herbivores search for plants and water.
Carnivores search for herbivores and water.
Creatures can die, reproduce, and move autonomously.

Controls:
- Use the control panel to change creature settings.
- Press Spawn to add creatures to the world.

Made with vanilla JavaScript.
